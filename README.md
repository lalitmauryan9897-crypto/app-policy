## 🚀 Quick Deploy (Choose One)

### Option 1: GitHub Pages (Easiest - 30 seconds)
1. Go to your repo: **Settings → Pages**
2. Under "Build and deployment", select:
   - Source: **Deploy from a branch**
   - Branch: **main**
3. Click **Save**
4. ✅ Your site is live at: `https://lalitmauryan9897-crypto.github.io/app-policy/privacy-policy.html`

### Option 2: Vercel (Auto-updates)
1. Go to: https://vercel.com/new
2. Select **app-policy** repository
3. Click **Deploy**
4. ✅ Site goes live instantly!

### Option 3: Netlify
1. Go to: https://app.netlify.com/start
2. Connect GitHub
3. Select this repo
4. ✅ Auto-deployed!
